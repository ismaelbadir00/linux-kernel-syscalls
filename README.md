\# linux-kernel-syscalls (Process Clearance)



Linux kernel programming project implementing custom system calls (x86-64) in a Linux 4.15.x kernel.

The feature adds a lightweight “process clearance/permissions” model and enforces access checks between processes.



\## Contents

\- `kernel/hw2.c` – kernel-space implementation

\- `docs/HW2.pdf` – assignment/specification



\## Skills demonstrated

\- Linux kernel development workflow

\- System call integration

\- Process/task handling and permission checks

\- Robust error handling and input validation
